Project:     Reality:Shard 
Info:        It's a generic game-server based on Java's servlets.
Languages:   Java
People:      _rusty (author)



Install notes for Developers:
-----------------------------
 - install git
 - install Java SDK 1.7
 - install maven3
 - set JDK 7 as the standard java environment
 - clone the repository (or fork it on github first)
 - get to know maven ;D
 - if you want to work with Eclipse or IdeaJ check out how
   to create the project settings files with the maven plugin
