Reality:Shard is a generic game-server based on Java's servlets.

Install notes for Developers:
 - get a latest version of the Java SDK 1.7
 - install eclipse for java developers, add the UMLet plugin
 - make sure you have set JDK 7 as the standart java environment
 - install git
 - clone the repository (or fork it on github first)
 - if you haven't, add it to your eclipse workspace
